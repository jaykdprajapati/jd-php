Please follow the steps below to install the code:

1. Copy and clone the code into your PC from this repository link: https://github.com/jaykdprajapati/jd-php.git

2. After completing the above step, open any terminal and type "composer install." This will download all required packages and files.

3. Create a database named 'jdphp.'

4. Run "php artisan migrate" to create all tables in the database.

5. Run "php artisan db:seed" to store the admin user in the database. The details are as follows:
   Email: adminjdphp@yopmail.com
   Password: password

6. Further development tasks will be handled by me.

7. Please refer to the code in the env.example file and replace it with your .env file.
