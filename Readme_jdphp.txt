Please follow below steps to install code.


1. Copy and clone the code into your pc from this repo link.

2. Once done above step, open any termial and type "composer install". Which will add download all require packages and files.

3. Please create one database and named it 'jdphp'.

4. Run "php artisan migrate" which will create all tables into the database.

5. Run "php artisan db:seed" to store admin user in database. details will as below.
    Email :   adminjdphp@yopmail.com
    Password : password

6. Rest will be the task developed by me.

7. Please consider env.example file code and replace it with you .env file.
